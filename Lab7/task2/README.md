# Task 2 - Object-Oriented Programming in Python

This project demonstrates Object-Oriented Programming in Python using an animal class hierarchy.

## Classes
- `Animal` - base class
- `Dog` - child class
- `Cat` - child class

## Implemented OOP concepts
- constructor `__init__`
- attributes
- instance methods
- `__str__`
- inheritance
- method overriding
- polymorphism
- modules and imports

## File structure
- `models.py` - contains all classes
- `main.py` - creates objects and demonstrates usage

## How to run
```bash
python main.py
