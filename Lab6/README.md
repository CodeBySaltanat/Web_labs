# Lab 6 — Album Browser (Routing & HTTP)

This project is an Angular application built for Lab 6 (Web Development course).

## Description
The application demonstrates:
- Angular Routing (multiple routes and route parameters)
- HttpClient for REST API requests
- CRUD operations (Read, Update, Delete)
- JSONPlaceholder API integration

The app allows users to:
- View a list of albums
- Open album details
- Update album title
- Delete albums
- View album photos

## API Used
JSONPlaceholder API  
https://jsonplaceholder.typicode.com

## How to Run the Project

1. Install dependencies:
npm install

2. Start development server:
ng serve

3. Open in browser:
http://localhost:4200
